# Short Codes

Short codes are **static or fixed codes** that serve as a gateway to Telehealth sessions for each campus or student. A **short code lobby** will display the first few upcoming sessions of the student or the campus.

|  |
| --- |
| **NOTE:** Although the URL formatting looks the same (ex: **https://telehealth.parallellearning.com/s/ABCDE**), **short codes are NOT the same as session codes.** Short codes have 6 digits, while session codes only have 5. |

When joining a session from a short code, you will not be prompted to log into a facilitator or student account, making it suitable for facilitators to use.

A user can also enter the campus/student short code here: [https://telehealth.parallellearning.com/join](https://telehealth.parallellearning.com/join "https://telehealth.parallellearning.com/join") to access the short code lobby.

### Campus short codes

* Campus short codes are entry points to Telehealth lobbies for each campus, enabling streamlined session access from schools
* Suitable for schools who have more than one facilitator in a campus, as well as facilitators who cannot log into their facilitator account every time to join a session

### Student short codes

* Suitable for students who are joining sessions on their own (from school or from home)

### How do I retrieve a campus or student short code?

Reach out to your Customer Success Manager and they'll be happy to provide you the short code you need.

### I don't see the session I'm looking for on my campus short code lobby

Please note upcoming sessions that show up on the short code lobby are dependent on the current time. If you think the upcoming session should be showing up on the lobby, please reach out to Parallel Support at [support@parallellearning.com](mailto:support@parallellearning.com) for technical assistance.
