# Joining sessions (gifs)

There are multiple ways that a facilitator is able to join a Parallel session within Pathway. Below, we will go over the different options you can use to access your sessions!

Click here to jump to our video walkthrough at the bottom of this article: [**Video Walkthrough**](https://help.parallellearning.com/hc/en-us/articles/39967286126107#h_01KES4QZSVCVY82XNHFKNB6WV2)!

# There are several ways to join a session:

* #### [**Joining via Pathway Calendar**](https://help.parallellearning.com/hc/en-us/articles/39967286126107#h_01K7501MTG5ZE1SRPPB6WEMYBH)
* #### [**Joining via Daily Session Summary Email**](https://help.parallellearning.com/hc/en-us/articles/39967286126107#h_01KC1S7JFDR25XBF6J7SSMHY7G)
* #### [**Joining via Session Code**](https://help.parallellearning.com/hc/en-us/articles/39967286126107#h_01K7502C82E4T0FSXPD1F3P4E4)
* #### [**Joining via Student Short Code**](https://help.parallellearning.com/hc/en-us/articles/39967286126107#h_01KES6B4Z3B6GNX27SJ6RXAA8J)
* #### [**Joining via Campus Short Code**](https://help.parallellearning.com/hc/en-us/articles/39967286126107#h_01KC1SWXWDPVPXAX3PHGY3VTK7)

#### **Joining via Pathway Calendar**

1. On the **Parallel platform** [**(app.parallellearning.com)**](https://app.parallellearning.com/), navigate to **Calendar.**

   Your calendar will look something like this:

   ![](https://help.parallellearning.com/hc/article_attachments/45448192828187)
2. Click on the upcoming session you wish to join on behalf of a student and click **Join Session**.

![](https://help.parallellearning.com/hc/article_attachments/45448192830363)![](https://help.parallellearning.com/hc/article_attachments/45448192831515)

You can also access your sessions via the List View of the calendar by clicking the video camera icon to the very right of the associated session. This is shown below.

![](https://help.parallellearning.com/hc/article_attachments/45448225725851)![](https://help.parallellearning.com/hc/article_attachments/45448192838043)

#### **Joining via Daily Session Summary Email**

1. Facilitators receive a **Daily Session Summary** in their registered email at the start of the day. From there, click the link for the corresponding session.

   |  |
   | --- |
   | **Note:** Each session has a dedicated link that will only work on the day of the session, so ensure you're using the correct link for the session you're trying to access. |

   ![](https://help.parallellearning.com/hc/article_attachments/45448225732123)![](https://help.parallellearning.com/hc/article_attachments/45448192841883)

   #### **Joining via Session Code**

   1. Go to [**https://telehealth.parallellearning.com/join**](https://telehealth.parallellearning.com/join) and enter the corresponding session code, which you will see on your Pathway calendar, or the Daily Session Summary email

   Below are examples of where you will see the session codes in both the Pathway calendar and the Daily Session Summary email:

   **Daily Session Summary Email:**

   ![](https://help.parallellearning.com/hc/article_attachments/45448192844187)

**Pathway Calendar:**

![](https://help.parallellearning.com/hc/article_attachments/45448225735451)

2. After locating the session code and navigating to [**https://telehealth.parallellearning.com/join**](https://telehealth.parallellearning.com/join) , all you need to do is enter your session code and you will be brought to your session room.

![](https://help.parallellearning.com/hc/article_attachments/45408190979995)![](https://help.parallellearning.com/hc/article_attachments/45448192850587)

|  |

| **Note:** We suggest bookmarking the link on the students' devices for easy access. If you encounter an error while joining a session, please email support@parallellearning.com immediately and inform the Provider about the issue. |

#### **Joining via Student Short Code**

1. To join via the Student Short Code, you must first navigate to the specific student's profile. To do this, you can select the Student tab in the top options bar and then select the name of the specific student. The student's short code will be found in their **Personal Info** section as shown below:

   ![](https://help.parallellearning.com/hc/article_attachments/45448192853531)

|  |
| --- |
| **Note:** The student short code is only associated with that specific student. Using this short code will only allow you to access sessions that include that student, and cannot be used to access other student sessions. |

2. Once you click on the student short code, the session waiting room will prompt you to select a session for that student. This waiting room will show any sessions the student is a part of that day.

   ![](https://help.parallellearning.com/hc/article_attachments/45448192857243)

![](https://help.parallellearning.com/hc/article_attachments/45448192860571)![](https://help.parallellearning.com/hc/article_attachments/45448225747867)

#### **Joining via Campus Short Code**

1. To join via the Campus Short Code, you must first navigate to **District 101** in Pathway. To do this, log into Pathway and select your name in the top right corner of the screen, then select **District 101** from the dropdown.

![Screenshot 2025-12-09 at 9.51.41 AM.png](https://help.parallellearning.com/hc/article_attachments/45408183270683)

2. Once you've navigated to your District 101 page, you will see a dropdown called **Campus Short Codes**. By selecting this, you will see the short codes for every campus associated with the students you facilitate for at your district.
3. To access sessions for all students you facilitate at the specific campuses in your dropdown, simply click on the short code linked to the associated campus.

![](https://help.parallellearning.com/hc/article_attachments/45448225749019)

|  |
| --- |
| **Note:** When using this short code you will only see the upcoming 3-4 sessions on the calendar scheduled for that day, not all sessions occurring that day. This list will update as your sessions are completed that day |
